# Exercise 1: Building an App with Lovable.dev

**Goal:** Build a functional web app using only natural language
**Requirements:** Browser, free Lovable.dev account

---

## What is Lovable.dev?

Lovable.dev is an AI-powered platform that creates full-stack web applications from natural language descriptions. You describe what you want, and the AI builds it ‚Äî complete with UI, database, and functionality.

**No coding required.** You just describe your idea like you would to a colleague.

---

## Getting Started

### Step 1: Create an Account
1. Go to [lovable.dev](https://lovable.dev)
2. Click **Sign Up** (or **Log In** if you already have an account)
3. Use your email or Google/GitHub authentication

### Step 2: Start a New Project
1. Click **Create New Project**
2. You'll see a chat interface where you can describe your app

### Describe Your App

Type a description like this in the chat:

> **Example prompt:**
> "Create a simple to-do list app where I can:
> - Add new tasks with a title and due date
> - Mark tasks as complete
> - Delete tasks I don't need

## Tips for Success

### ‚úÖ Do This

- **Be specific about features** ‚Äî "Add a due date" is better than "make it better"
- **Describe the user experience** ‚Äî "When I click the button, show a success message"
- **Mention design preferences** ‚Äî "Use a minimalist design" or "Make it colorful and fun"
- **Iterate in small steps** ‚Äî Add one feature at a time

### ‚ùå Avoid This

- Describing the entire app in one message can make things harder to track
- Don't get too frustrated if it doesn't turn out right ‚Äî just ask for changes or start again

## Example Conversation

Here's what a typical session might look like:

**You:** "Create a birthday reminder app where I can save birthdays with the person's name and date. Show me upcoming birthdays in the next 30 days."

**Lovable:** *Builds initial app with a form and list*

**You:** "Show how many days until each birthday"
